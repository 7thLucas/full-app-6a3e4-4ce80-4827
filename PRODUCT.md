# PropLedger

## Overview
Property expense management app for property managers and owners with large portfolios (500+ units).

## Users
- Primary: Property owner/manager who personally manages the portfolio
- Scale: 500+ units across multiple properties
- Revenue is already handled externally by an accountant — this app focuses entirely on expense tracking and allocation

## Core Problem
Currently, allocating expenses to the correct unit requires scrolling through all 500+ units — slow, error-prone, and frustrating. PropLedger solves this with instant unit lookup.

## Core Workflow
1. Log an expense: enter amount + unit number → expense is instantly allocated to that specific unit
2. Fast unit search/lookup — no scrolling through hundreds of units
3. Monthly P&L view per property — see income vs expenses for each property each month

## Key Features (MVP)
- Expense entry form: amount, unit number, category, date, notes
- Unit search: type a unit number/name to find it instantly
- Monthly P&L dashboard: per-property breakdown of income vs expenses
- Property list: manage multiple properties with their units
